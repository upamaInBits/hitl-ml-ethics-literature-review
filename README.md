# Ethical Analysis of Human-in-the-Loop Machine Learning

## Overview
This project is a systematic literature review analyzing how ethical principles are discussed in Human-in-the-Loop (HITL) Machine Learning research.

While HITL systems are widely used to improve machine learning performance, their ethical implications are often overlooked.

---

## Key Insight
Through analysis of 74 research papers, this study found that:

- Only ~26% of papers included meaningful discussions of ethical principles
- Ethics discussions were inconsistent and often limited to specific domains

---

## Research Questions
- How frequently are ethical principles discussed in HITL ML literature?
- Which ethical principles are most commonly addressed?
- In what application domains are these principles applied?

---

## Methods
- Systematic literature review
- Source: ACM Digital Library
- Papers analyzed: 74
- Data extracted using keyword-based and manual review methods

---

## Key Findings
- Most discussed principles:
  - Fairness
  - Transparency
  - Explainability
  - Interpretability
- Many important principles (e.g., safety, human dignity) were rarely or never discussed
- Ethical discussions are not yet standardized in ML research
